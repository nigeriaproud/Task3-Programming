## 計算の定義

### 未参加人数の計算 (task4.c)
クラス人数 $a$ 人、野球(9人)を $b$ チーム、バレー(6人)を $c$ チームとした時、参加できない人数を算出します。
$$\text{未参加人数} = a - (9b + 6c)$$

### 三角形の面積 (task5.c)
底辺を $w$、高さを $h$ とした時の面積を算出します。
$$\text{面積} = \frac{1}{2}wh$$
